O que é HTML?

HTML é um editor de hipertextos, muito utilizado para criação de páginas online e aplicações de web. Em conjunto com o CSS e Javascript, eles formam as pedras principais para a World Wide Web. Todos os navegadores atuais recebem documentos em HTML que são processados renderização e apresentação do conteúdo online. O nome HTML é uma abreviação para a expressão inglesa de HyperText Markup Language, que significa Linguagem de Marcação de Hipertexto. A tecnologia é fruto da junção entre os padrões HyTime e SGML, que não são mais utilizados.
